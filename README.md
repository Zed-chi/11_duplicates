# Anti-Duplicator

Prints duplicate files in provided path
## Usage

`python duplicate.py -p <some_directory>`

Example in Windows:

```bash
python duplicate.py -p C:\work

two.txt:

   work/two.txt
   work/example/sub/two.txt

 one.txt:

   work/one.txt
   work/example/one.txt
```
